# easycallcenter365

![easycallcenter365](logo.jpg)

基于FreeSWITCH和大模型的智能电话客服系统。 当前项目是 `easycallcenter365` 的web管理界面。

### 功能列表

* easycallcenter365 的参数在线配置
* 电话工具条
* FreeSWITCH在线配置
* 分机管理
* 日志在线监控
* 语音识别配置
* 语音合成配置
* 通话记录查询
